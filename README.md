# HormigApp IA

## Descripción
HormigApp IA es un prototipo beta orientado a la segmentación de gastos hormiga mediante aprendizaje no supervisado.

## Objetivo General de la Beta
Establecer una base funcional inicial para organizar datos de gastos y preparar el flujo de trabajo para clasificación por clusters en futuras iteraciones.

## Tecnologías Previstas
- Python
- Streamlit
- Pandas
- scikit-learn
- Matplotlib
- NumPy

## Instalación y Ejecución (Inicial)
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

> Nota: Esta versión solo contiene la estructura base del proyecto.
